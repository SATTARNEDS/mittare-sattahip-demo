# Mittare Sattahip — Demo Website

Static responsive landing page สำหรับเดโมทีม Mittare Sattahip

## เปิดใช้งาน

เปิด `index.html` ด้วยเบราว์เซอร์ หรือใช้ local server:

```powershell
python -m http.server 8080
```

จากนั้นเปิด `http://localhost:8080`

## หมายเหตุ

- แบบฟอร์มเป็น UI demo และยังไม่ส่งข้อมูล
- เมนูเช็กเบี้ย DIY เป็นสูตรประมาณการสำหรับสาธิต UX ไม่ใช่อัตราเบี้ยหรือใบเสนอราคาของบริษัท
- ระบบคำนวณใหม่ใช้ยอดจริงเฉพาะ พ.ร.บ. ซึ่งมีอัตราสาธารณะ โดยคำนวณเบี้ยสุทธิ อากรแสตมป์ และ VAT แยกรายการ
- แผนสมัครใจและแผนทรัพย์สิน/บุคคล/ธุรกิจที่ไม่มีฐานเบี้ยสาธารณะ จะสร้างใบขอคำนวณเบี้ยแทนการสร้างตัวเลขประมาณการ
- เงื่อนไขตัวเลขที่บริษัทเผยแพร่ เช่น ส่วนลดมิตรแท้เพิ่มพูน 2+ และค่า Wrap/Sticker บางแผน จะแสดงในใบสรุปประกอบการขอราคา
- ใบสรุปใช้ Print Dialog ของเบราว์เซอร์ เลือก `Save as PDF` เพื่อบันทึกไฟล์ PDF
- รายละเอียดผลิตภัณฑ์เปิดอ่านภายในเว็บไซต์ ไม่พาผู้ใช้ออกจากหน้า Demo
- ศูนย์ข้อมูลประกันมี 29 แผน/ประเภท แยกเมนูรถยนต์ บ้านและทรัพย์สิน บุคคล ธุรกิจ และประกันเฉพาะทาง พร้อมช่องค้นหา
- เมนูหมวดประกันอยู่บน Navigation หลัก และเปิดหน้า `insurance.html` ด้วย URL แยกตามหมวด เช่น `insurance.html?category=motor`
- หน้าแรกแสดงภาพและทางลัดไปยังแต่ละหมวด ส่วนหน้าหมวดแสดงภาพประกอบบนทุกแผนเพื่อช่วยแยกบริบท
- แต่ละแผนมีข้อมูลผู้ที่เหมาะสม ความคุ้มครองหลัก ปัจจัยกำหนดเบี้ย ข้อยกเว้นทั่วไป และเอกสารที่ควรเตรียม
- เบอร์โทรศัพท์, LINE และ Facebook ของทีมเป็นข้อมูล Demo ต้องเปลี่ยนก่อนนำเว็บไซต์ไปใช้งานจริง
- ข้อมูลผลิตภัณฑ์และช่องทางบริการอ้างอิงเว็บไซต์ `mittare.com` ณ วันที่ 19 มิถุนายน 2569
- ตัวเลขเบี้ยจริงไม่ได้กำหนดตายตัวในศูนย์ข้อมูล เนื่องจากต้องคำนวณจากข้อมูลผู้เอาประกัน ทรัพย์สิน/รถ ทุนประกัน และเงื่อนไขรับประกันภัย
- หน้าเว็บไซต์แสดงเฉพาะภาพบุคคลสองท่าน โดยใช้ไฟล์แยกจากกันและสร้างจากภาพอ้างอิงที่ผู้ใช้มอบให้ด้วย built-in image generation
- ก่อนเผยแพร่จริงควรตรวจสอบสิทธิ์การใช้ตราสัญลักษณ์ ชื่อทางการ ที่อยู่สำนักงานตัวแทน และข้อมูลติดต่อเฉพาะทีม

## Performance

ภาพบุคคลและภาพประกอบประกันมีขนาดค่อนข้างใหญ่เพื่อคงรายละเอียดบนจอใหญ่ หากนำขึ้น Production ควรแปลงเป็น AVIF/WebP หลายขนาดและใช้ `<picture>` เพื่อลดเวลาโหลดบนมือถือ
